Design Patterns Project
		Zoo Veterinar

	Este timpul pentru inspecția anuală a animalelor din zoo. 
	Veterinarul are sarcina de a lua fiecare animal la rând, de a-l consulta, de a-i da un diagnostic
și, unde este cazul,să trateze animalul bolnav. În proiectul respectiv am dezvoltat un program în care sunt
înregistrate animale, sunt supuse inspecției anuale, iar în funcție de problema fiecăruia, se va aplica
strategia de tratament, cu pașii corespunzători.

Design Patterns folosite:

Builder -> Pentru instanțierea animalelor
Strategy -> Pentru alegerea tratemenului unui animal
Template Methods -> Pașii pentru realizarea fiecărei strategii
